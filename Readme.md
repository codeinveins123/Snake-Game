# 🐍 SNAKE GAME by codeinveins  

---

## 🎮 How It Looks  

![Gameplay](game.gif)  

---

A state management system has been implemented, including the main menu, gameplay, game over, and pause states.
All assets were fully hand-drawn, and rendering is handled using SFML.
My game features three difficulty levels that add walls to the playing field. There are also buffs and debuffs — at the moment, they temporarily slow down or speed up the snake. Their presence replaces regular food, adding variety and dynamics to the gameplay.

---

## 🎮 Controls  

| Key | Action |
|-----|--------|
| **W** / **⬆ UpArrow** | Move Up |
| **A** / **⬅ LeftArrow** | Move Left |
| **S** / **⬇ DownArrow** | Move Down |
| **D** / **➡ RightArrow** | Move Right |
| **Esc** / **Space** | Pause |
| **Enter** / **Space** | Press menu button |
